Hexo踩过的坑汇总

目录

1.Develop not found
2.CyanStyle 无法正常显示背景图片（且Git push 同步已经是 updated
3.使用CyanStyle 时间错误

　　1.Develop not found

　　　　原因：没有Develop元件。
　　　　解决方法：通过npm下载

　　npm install hexo-deployer-git –save

　　2.CyanStyle 无法正常显示背景图片（且Git push 同步已经是updated)

　　　　原因：背景文件及其他一些不存在
　　　　解决方法：到GitHub网站上下载zip压缩包，解压然后覆盖原文件夹
　　　　网址

3.使用CyanStyle 时间错误

　　　　出现情况：每次post到GitHub.io 所有文件时间都会同步至post时间
　　　　原因：主题原因
　　　　解决方法：换主题
　　　　官方主题网址
